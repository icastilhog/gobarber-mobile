## Getting started

```bash
# Clone this repository
$ git clone https://github.com/icastilhog/gobarber-mobile

# Go into the repository
$ cd gobarber-mobile

# Install dependencies
$ yarn

# Run the app (iOS)
$ yarn ios

# Run the app (Android)
$ yarn android
```
